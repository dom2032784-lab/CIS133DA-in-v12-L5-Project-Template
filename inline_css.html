Generic Fonts

A font family, often called a typeface, describes the overall style and design of characters on a page. Fonts are organized into general categories based on shared features. Serif fonts have small finishing strokes at the ends of letters, which often give them a more classic or formal feel. Sans-serif fonts remove those strokes, resulting in a simpler and more modern look. Cursive fonts are created to imitate handwriting and may appear flowing or script-like. Fantasy fonts are decorative and stylized, typically used to add personality or creativity to a design. Most of these font families are proportional, meaning wider letters naturally take up more space than thinner ones. Monospace fonts differ because each character, regardless of shape, occupies the same amount of horizontal space.

Identifying Color

In HTML, color can be added to a webpage using CSS either within a style section or directly inside an element through a style attribute. One simple way to apply color is by using a standard color name such as blue, red, or green, which will change the appearance of the text to that specific color. Another common method is using hexadecimal, or hex, color codes. Hex colors begin with a hashtag symbol followed by six letters and numbers that represent different levels of red, green, and blue. For example, the code FF0000 creates red, while 000000 creates black. Color names are easier to remember, but hex codes provide more precise control over the exact shade displayed on a webpage.
